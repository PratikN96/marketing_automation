# YouTube Marketing Automation

An automated pipeline that scrapes viral video hooks, stitches a custom CTA, and schedules them to a YouTube channel — fully hands-off after the initial setup.

## How It Works

```
Scrape hooks  →  Stitch CTA  →  Upload to GitHub Release  →  GitHub Actions publishes daily
```

1. **Scrape** — Downloads the first 3 seconds of recent videos from a YouTube Shorts channel
2. **Stitch** — Appends your custom CTA clip to each hook
3. **Release** — Uploads final videos to GitHub Releases as storage
4. **Schedule** — GitHub Actions runs daily and uploads 3 videos to YouTube automatically

## Scripts

| Script | Purpose | When to run |
|---|---|---|
| `scrape_hooks.py` | Scrape hooks from a channel | Weekly (local) |
| `stitch_cta.py` | Stitch CTA to hooks | Weekly (local) |
| `release_upload.py` | Push videos to GitHub Release + update queue | Weekly (local) |
| `upload.py` | One-shot manual upload to YouTube | As needed |
| `daily_upload.py` | Upload 3 videos/day to YouTube | GitHub Actions (automatic) |
| `test_pipeline.py` | End-to-end pipeline test | Before going live |

## Weekly Workflow

```bash
# Step 1 — Scrape hooks
python scrape_hooks.py --channel "<channel_url>" --count 25

# Step 2 — Stitch CTA (place your cta.mp4 in the project root first)
python stitch_cta.py --hooks hooks/ --cta cta.mp4 --output final/ --limit 21

# Step 3 — Push to GitHub Release and schedule
python release_upload.py --folder final/ --start-date YYYY-MM-DD --skip 10
```

After Step 3, GitHub Actions handles daily uploads automatically. Local `hooks/` and `final/` folders can be deleted.

## Setup

### Requirements
- Python 3.10+
- ffmpeg (`brew install ffmpeg`)
- `pip install -r requirements.txt`

### YouTube API
1. Create a free project at [console.cloud.google.com](https://console.cloud.google.com)
2. Enable **YouTube Data API v3**
3. Create **OAuth 2.0 credentials** (Desktop app) → download as `credentials.json`
4. Run `python get_token.py` to authenticate and generate `token.json`

### GitHub Secrets
Add these to your repo under Settings → Secrets → Actions:

| Secret | Description |
|---|---|
| `YOUTUBE_CLIENT_ID` | From Google Cloud OAuth credentials |
| `YOUTUBE_CLIENT_SECRET` | From Google Cloud OAuth credentials |
| `YOUTUBE_REFRESH_TOKEN` | Generated by `get_token.py` |

### Configuration
Edit `config.py` to set your channel handle, video tags, publish times, and other preferences.

## Architecture

- **Storage** — GitHub Releases (free, no size limits per file up to 2GB)
- **State** — `queue.json` tracks pending and uploaded videos
- **Scheduler** — GitHub Actions cron (runs daily at 12pm UTC)
- **Quota** — 3 uploads/day = 4,800 API units (well within YouTube's free 10,000/day limit)
